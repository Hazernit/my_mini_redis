# Задание по проектной практике

## Основное задание

В рамках проектной практики необходимо подготовить учебный проект, оформить репозиторий, создать документацию, описать ход работы и подготовить демонстрацию результата.

Проект должен содержать:

- исходный код;
- описание проекта;
- демонстрационный сайт;
- документацию;
- отчетные материалы;
- инструкцию по запуску.

## Вариативная часть задания

Вариативная часть задания — практическая реализация технологии.

Выбранная тема: создание собственного Mini Redis Server на Python.

## Содержание вариативной части

В рамках вариативной части реализуется учебное key-value хранилище, работающее по TCP. Проект демонстрирует, как клиент отправляет команду серверу, сервер обрабатывает команду и возвращает результат.

Реализуемые компоненты:

- TCP Mini Redis Server;
- CLI-клиент;
- HTTP API для браузерной демонстрации;
- статический сайт проекта;
- страница демонстрации работы команд.

## Поддерживаемые команды

- `SET key value` — сохранить значение;
- `GET key` — получить значение;
- `DELETE key` — удалить значение;
- `EXISTS key` — проверить наличие ключа.

## Ожидаемый результат

Итогом работы является репозиторий с исходным кодом, документацией и демонстрационным сайтом. Проект должен запускаться локально и показывать работу Mini Redis Server через веб-интерфейс.
