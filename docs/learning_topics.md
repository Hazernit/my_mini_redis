# Темы для изучения

Для понимания проекта Mini Redis Server полезно изучить следующие темы.

## Python

- базовый синтаксис Python;
- словари и работа с данными в памяти;
- функции и обработка ошибок;
- запуск Python-скриптов из терминала;
- модуль `argparse` для параметров командной строки.

## Сетевое программирование

- клиент-серверная архитектура;
- TCP-сокеты;
- адрес, порт и подключение;
- отправка и получение данных;
- обработка нескольких клиентов;
- кодирование строк в байты и обратно.

## Redis и key-value хранилища

- принцип хранения данных в формате ключ-значение;
- команды `SET`, `GET`, `DELETE`, `EXISTS`;
- хранение данных в оперативной памяти;
- отличие учебной реализации от промышленной системы Redis.

## HTTP API

- протокол HTTP;
- методы `GET`, `POST`, `OPTIONS`;
- формат JSON;
- заголовок `Content-Type`;
- CORS и доступ браузера к API;
- роль API как прослойки между сайтом и TCP-сервером.

## Веб-разработка

- структура HTML-страницы;
- стилизация через CSS;
- обработка событий в JavaScript;
- отправка запросов через `fetch`;
- отображение результата на странице.

## Размещение проекта

- статический сайт и его публикация;
- запуск backend-приложения на сервере;
- переменные окружения;
- сетевые порты и правила доступа;
- базовые сервисы Yandex Cloud: Object Storage, Compute Cloud, Virtual Private Cloud и Security Groups.
